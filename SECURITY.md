# Security Policy

This repository inherits the security policy of the parent
[djehuty](https://github.com/4TUResearchData/djehuty) project.

See: <https://github.com/4TUResearchData/djehuty/blob/main/SECURITY.md>

## Scope

The policy above covers reporting and disclosure. Vulnerabilities specific
to the *Helm chart sources* in this repository (templates, default values,
CI/release tooling) may also be reported via a
[private security advisory](https://github.com/4TUResearchData/helm-charts/security/advisories/new)
on this repository — please **do not** file a public issue.

Vulnerabilities in the *upstream applications* packaged by these charts
(djehuty itself, Virtuoso, …) should be reported to their respective
projects.
